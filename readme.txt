Adds descriptions to product options of the "Product Options for WooCommerce" plugin.


== Description ==

This plugin requires [Simple Product Options for WooCommerce](https://wordpress.org/plugins/product-options-for-woocommerce/) plugin.

Allows to add notes and descriptions to product options. So the customer can read extra information about the option or specific option value.
It is possible to use HTML formating and links in the description text.


You can check a demo on this page:
**[DEMO Page](http://hottons.com/demo/wp/ef/product/option-descriptions/)**

You can read the installation instructions here:
**[README](http://hottons.com/demo/wp/ods/README.html)**


**Simple Code Idea:**

The idea of this plugin is to keep the base version as simple as possible.
And to add the new features as modifications.

This plugin has just 18 files to make it easy to use and customize. If you need more features check the modifications page [hottons.com/ods_modifications](http://hottons.com/ods_modifications)

== Installation ==

Requires  WordPress 4.7,  WooCommerce 3.0  and  Simple Product Options for WooCommerce 1.0

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `POFW Option Description` from Plugins page.

Contact me by email <pektsekye@gmail.com> if you have questions or need help.
